    public crWin(win: undefined, htmlFile: string, frame: boolean, icon: string, devTools: boolean, width: number, height: number, name: string): object source

    Create new browser window (Main process)

    Params:

    NameTypeAttributeDescription
    win undefined

    Variable which will contain BrowserWindow object
    htmlFile string

    Relative path to file to load
    frame boolean

    Specifies whether the window frame is displayed
    icon string

    Relative path to icon for a window
    devTools boolean

    Specifies whether the developer tools are displayed when the window is opened
    width number

    Width of a window
    height number

    Height of a window
    name string

    Name of a window

    Return:

    object

    Object containing opened window

    Example:

    <JavaScript>
mainWindow = crWin(mainWindow, 'index.html', true, 'res/icon.ico', false, 800, 600, 'Main Window')
